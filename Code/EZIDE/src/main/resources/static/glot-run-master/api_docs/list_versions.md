## Listing versions

##### List versions
    curl --request GET \
         --url 'https://run.glot.io/languages/python'

### Example response
    [
      {
        "url": "https://run.glot.io/languages/python/2",
        "version": "2"
      },
      {
        "url": "https://run.glot.io/languages/python/latest",
        "version": "latest"
      }
    ]
